=======================
Access to the databases
=======================


Direct SQL access using Docker
==============================

#. Use a GUI like HeidiSQL or SequelPro
#. Be sure that your local docker is running
#. The server is accessible with the credentials located in :file:`.env`
#. Access via port `3310` (t3o) or `3311` (ter) to `localhost`

Have fun!
